Build a Racecar

Practice project utilizing HTML, CSS, an SQLite database, and Node.js backend. The site uses Bootstrap CSS.

Current features: Add items to database / Show all items in database // Show installed parts / Show Car Stats / Update car stats after part purchase / Show budget // Update budget after purchase.

Features and changes to be done: Creating a preset of parts // Limiting parts per category // Setting tresholds and indicators for too high emissions and too low handling etc.

The game idea is that you can upgrade your car using performance parts. You have a limited budget. Try to keep the emissions within limits while increasing power without sacrificing too much handling. Good luck!
